Tables

Lists

An HTML table is defined with the ^table^ tag.
Each table row is defined with the ^tr^ tag.
A table header is defined with the ^th^ tag.
By default, table headings are bold and centered.
A table data/cell is defined with the ^td^ tag.
To add a table name, use the ^caption^ tag.

Lists

  • An unordered list starts with the UL tag
  • Each list item starts with the LI tag.
  • List items will be marked with bullets by default
  • The CSS list-style-type property is used to define list markers
  • Style Types:
    • Disc
    • Circle
    • Square
    • None
  1. An ordered list starts with the ol tag
  2. Each item in the list starts with the li tag
  3. The list items will be marked with numbers by default:
  4. the type attribute of the ol tag, defines the type of a list item markers
  5. List Types:
    1. type="1" - Numbered
    2. type="A" - Uppercase Letters
    3. type="a" - Lowercase Letters
    4. type="I" - Upercase Roman Numerals
    5. type="i" - Lowercase Roman Numerals
